문자열로 기록된 표현식을 읽어들여 원래의 Python 데이터를 얻어내는 방법.

예)
    data = '[1,2,3,4,5]'

ast.literal_eval()은 위 예를 읽어들여 그에 해당하는 Python 데이터를 리턴한다.

    print(ast.literal_eval(data))

ast.literal_eval()은 표현식을 실행하기 전 안전성을 먼저 검증하므로 보다 안전합니다.
